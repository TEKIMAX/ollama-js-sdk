# tekimax-sdk learn

Launch the interactive AI Academy to learn about AI, LLMs, and machine learning concepts.

## Usage

```bash
tekimax-sdk learn
```

## Description

The `learn` command opens an interactive terminal-based learning platform with:
- 5 comprehensive chapters
- Interactive navigation
- Quizzes with immediate feedback
- Searchable content
- Quick reference guides

## Features

### Interactive Menu System

Navigate through chapters and lessons using:
- **Arrow Keys**: Move up/down through options
- **Enter**: Select an item
- **B**: Go back
- **S**: Search for topics
- **Q**: Access quick reference
- **X**: Exit

## Troubleshooting

<Accordion title="Terminal colors not showing">
  Some terminals may not support colors. Try:
  - Using a modern terminal (iTerm2, Windows Terminal)
  - Setting `FORCE_COLOR=1` environment variable
  - Using a different shell (zsh, bash)
</Accordion>

<Accordion title="Navigation keys not working">
  If arrow keys don't work:
  - Use number keys to select options
  - Press Enter after typing the number
  - Try a different terminal emulator
</Accordion>

## Exit and Resume

Your progress is not saved between sessions, but you can:
- Exit anytime with 'X' or Ctrl+C
- Resume by running `tekimax-sdk learn` again
- Jump to specific chapters using the menu
