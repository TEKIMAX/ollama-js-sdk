## Model Comparison

| Feature | GPT-OSS 20 | GPT-OSS 120 |
|---------|------------|-------------|
| **Parameters** | 20B | 120B |
| **Context Window** | 8,192 tokens | 32,768 tokens |
| **Max Output** | 4,096 tokens | 16,384 tokens |
| **Speed** | Fast | Slower |
| **Quality** | Good | Excellent |
| **Use Cases** | General tasks | Complex reasoning |

## Quick Examples

<Tabs>
  <Tab title="Generate Text">
    ```typescript
    import { OllamaClient } from 'tekimax-sdk';

    const client = new OllamaClient();
    
    const response = await client.models.generate({
      model: 'gpt-oss-20',
      prompt: 'Explain quantum computing',
      temperature: 0.7
    });
    
    console.log(response.response);
    ```
  </Tab>
  <Tab title="Create Embeddings">
    ```typescript
    const embedding = await client.embeddings.create({
      model: 'gpt-oss-20',
      input: 'Convert this text to vectors'
    });
    
    console.log(embedding.embeddings[0]);
    ```
  </Tab>
  <Tab title="CLI Usage">
    ```bash
    # Generate text
    tekimax-sdk generate -m gpt20 -p "Hello, AI!"
    
    # Create embeddings
    tekimax-sdk embed -m gpt-oss-20 -p "Semantic search"
    
    # Check setup
    tekimax-sdk setup
    ```
  </Tab>
</Tabs>
